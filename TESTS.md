# Конфигурация запуска тестов

### Максимум 4 параллельных теста
`go test -parallel=4 ./...`

### 2 параллельных компиляции пакетов
`go test -p=2 ./...`

### Ограничение использования CPU
`go test -cpu=2 ./...`

### Таймаут для тестов
`go test -timeout=30s ./...`

### Пропускать кэширование
`go test -count=1 ./...`
